🔍 Verificación de IDs de Rutina

📋 Cómo usar esta herramienta

Esta herramienta te ayudará a diagnosticar problemas con los IDs de rutina en TrainFit y verificar por qué el modal no se abre.

Paso 1: Navegar a la página del cliente

Ve a la página de progreso del cliente donde están los botones "Ver Detalles" que no funcionan:

http://localhost:5173/trainer/clients/[CLIENT_ID]/progress

Paso 2: Abrir las herramientas de desarrollador

Presiona F12 o Ctrl+Shift+I (Cmd+Option+I en Mac) para abrir las herramientas de desarrollador.

Paso 3: Ir a la pestaña Console

Haz clic en la pestaña "Console" en las herramientas de desarrollador.

Paso 4: Copiar y pegar el script

Ve a la pestaña "Script" de esta herramienta, copia todo el código y pégalo en la consola del navegador.

Paso 5: Presionar Enter

Presiona Enter para ejecutar el script. El diagnóstico se ejecutará automáticamente.

✅ Qué esperar

  • El script verificará automáticamente los IDs de rutina
  • Interceptará las llamadas a la API para ver los datos
  • Simulará un clic en "Ver Detalles" para probar el modal
  • Detectará errores en la consola
  • Proporcionará información detallada sobre el problema

⚠️ Funciones adicionales

Después de ejecutar el script, tendrás acceso a:

debugRoutine("ID_DE_RUTINA")

Esta función te permite verificar una rutina específica por su ID.

✨ Características del Script de Diagnóstico

🔍 Verificación de Estado

Analiza el estado de React para encontrar datos de rutinas y detectar problemas en el componente.

🌐 Interceptor de API

Intercepta todas las llamadas a la API de rutinas para verificar qué datos se están recibiendo.

🖱️ Simulación de Clics

Simula automáticamente clics en los botones "Ver Detalles" para probar la funcionalidad del modal.

📊 Análisis del DOM

Examina el DOM en busca de elementos relacionados con rutinas y botones de acción.

❌ Detección de Errores

Intercepta y registra todos los errores de JavaScript que puedan estar causando el problema.

🛠️ Herramientas de Debug

Proporciona funciones adicionales para verificar rutinas específicas manualmente.

📈 Interpretación de Resultados

  • ✅ Verde: Funcionamiento correcto
  • ⚠️ Amarillo: Advertencias o datos faltantes
  • ❌ Rojo: Errores que requieren atención
  • 🔍 Azul: Información de diagnóstico

⚠️ Importante: Copia todo el código de abajo y pégalo en la consola del navegador mientras estás en la página del cliente.

// Script para verificar IDs de rutina y validar datos // Ejecutar en la consola del navegador en la página TrainerClientProgressPage (function() { console.log('🔍 === INICIANDO VERIFICACIÓN DE IDs DE RUTINA ==='); // Función para obtener el clientId de la URL function getClientIdFromUrl() { const path = window.location.pathname; const match = path.match(/\/trainer\/clients\/([^/]+)\/progress/); return match ? match[1] : null; } // Función para verificar si hay rutinas en el estado de React function checkReactState() { console.log('🔍 Verificando estado de React...'); // Buscar el componente React en el DOM const reactRoot = document.querySelector('#root'); if (!reactRoot) { console.error('❌ No se encontró el elemento root de React'); return null; } // Intentar acceder al estado a través de React DevTools const reactFiber = reactRoot._reactInternalFiber || reactRoot._reactInternalInstance; if (reactFiber) { console.log('✅ Componente React encontrado'); return reactFiber; } console.log('⚠️ No se pudo acceder al estado de React directamente'); return null; } // Función para verificar rutinas en el DOM function checkRoutinesInDOM() { console.log('🔍 Verificando rutinas en el DOM...'); const routineButtons = document.querySelectorAll('button:contains("Ver Detalles")'); console.log(`📊 Botones "Ver Detalles" encontrados: ${routineButtons.length}`); // Buscar tarjetas de rutina const routineCards = document.querySelectorAll('[class*="routine"], [class*="Routine"]'); console.log(`📊 Tarjetas de rutina encontradas: ${routineCards.length}`); // Buscar texto que contenga "Rutina" const routineTexts = document.querySelectorAll('*'); let routineCount = 0; routineTexts.forEach(el => { if (el.textContent && el.textContent.includes('Rutina') && el.children.length === 0) { console.log(`📝 Texto de rutina: ${el.textContent.trim()}`); routineCount++; } }); return { routineButtons: routineButtons.length, routineCards: routineCards.length, routineTexts: routineCount }; } // Función para interceptar llamadas a la API function interceptAPICall() { console.log('🔍 Configurando interceptor de API...'); const originalFetch = window.fetch; window.fetch = function(...args) { const url = args[0]; if (typeof url === 'string' && url.includes('/routines')) { console.log('🌐 Interceptada llamada a API de rutinas:', url); return originalFetch.apply(this, args) .then(response => { console.log('📡 Respuesta de API de rutinas:', response.status, response.statusText); // Clonar la respuesta para poder leerla const clonedResponse = response.clone(); clonedResponse.json().then(data => { console.log('📊 Datos de rutinas recibidos:', data); if (data && Array.isArray(data)) { console.log(`✅ ${data.length} rutinas encontradas`); data.forEach((routine, index) => { console.log(`📋 Rutina ${index + 1}:`, { id: routine.id, name: routine.name, clientId: routine.clientId, exercises: routine.exercises ? routine.exercises.length : 'N/A' }); }); } else if (data && data.data && Array.isArray(data.data)) { console.log(`✅ ${data.data.length} rutinas encontradas en data.data`); data.data.forEach((routine, index) => { console.log(`📋 Rutina ${index + 1}:`, { id: routine.id, name: routine.name, clientId: routine.clientId, exercises: routine.exercises ? routine.exercises.length : 'N/A' }); }); } else { console.log('⚠️ Formato de datos inesperado:', data); } }).catch(err => { console.error('❌ Error al parsear respuesta JSON:', err); }); return response; }) .catch(error => { console.error('❌ Error en llamada a API de rutinas:', error); throw error; }); } return originalFetch.apply(this, args); }; console.log('✅ Interceptor de API configurado'); } // Función para simular clic en botón "Ver Detalles" function simulateButtonClick() { console.log('🔍 Buscando botones "Ver Detalles"...'); // Buscar todos los botones que contengan "Ver Detalles" const buttons = Array.from(document.querySelectorAll('button')).filter(btn => btn.textContent && btn.textContent.trim().includes('Ver Detalles') ); console.log(`📊 Botones "Ver Detalles" encontrados: ${buttons.length}`); if (buttons.length > 0) { console.log('🖱️ Simulando clic en el primer botón "Ver Detalles"...'); // Agregar listener para detectar cambios en el DOM const observer = new MutationObserver((mutations) => { mutations.forEach((mutation) => { if (mutation.type === 'childList') { mutation.addedNodes.forEach((node) => { if (node.nodeType === 1) { // Element node if (node.classList && (node.classList.contains('modal') || node.querySelector && node.querySelector('.modal'))) { console.log('✅ Modal detectado en el DOM:', node); // Verificar contenido del modal const modalContent = node.textContent || ''; console.log('📝 Contenido del modal:', modalContent.substring(0, 200) + '...'); observer.disconnect(); } } }); } }); }); observer.observe(document.body, { childList: true, subtree: true }); // Hacer clic en el botón buttons[0].click(); // Desconectar observer después de 5 segundos setTimeout(() => { observer.disconnect(); console.log('⏰ Observer desconectado después de 5 segundos'); }, 5000); } else { console.log('❌ No se encontraron botones "Ver Detalles"'); } } // Función para verificar errores en consola function checkConsoleErrors() { console.log('🔍 Verificando errores en consola...'); // Interceptar console.error const originalError = console.error; const errors = []; console.error = function(...args) { errors.push(args); originalError.apply(console, args); }; // Restaurar después de 10 segundos setTimeout(() => { console.error = originalError; if (errors.length > 0) { console.log('❌ Errores detectados:', errors); } else { console.log('✅ No se detectaron errores en consola'); } }, 10000); } // Función principal function runDiagnostic() { const clientId = getClientIdFromUrl(); console.log('🆔 Client ID desde URL:', clientId); if (!clientId) { console.error('❌ No se pudo obtener el Client ID de la URL'); return; } // Verificar estado de React checkReactState(); // Verificar rutinas en DOM const domCheck = checkRoutinesInDOM(); console.log('📊 Resumen DOM:', domCheck); // Configurar interceptor de API interceptAPICall(); // Verificar errores en consola checkConsoleErrors(); // Simular clic después de 2 segundos setTimeout(() => { simulateButtonClick(); }, 2000); console.log('✅ Diagnóstico iniciado. Revisa los logs en los próximos segundos...'); } // Ejecutar diagnóstico runDiagnostic(); // Función para verificar manualmente una rutina específica window.debugRoutine = function(routineId) { console.log(`🔍 Verificando rutina específica: ${routineId}`); // Hacer llamada directa a la API const token = localStorage.getItem('token'); if (!token) { console.error('❌ No se encontró token de autenticación'); return; } fetch(`/api/trainer/routines/${routineId}`, { headers: { 'Authorization': `Bearer ${token}`, 'Content-Type': 'application/json' } }) .then(response => { console.log('📡 Respuesta para rutina específica:', response.status); return response.json(); }) .then(data => { console.log('📊 Datos de rutina específica:', data); }) .catch(error => { console.error('❌ Error al obtener rutina específica:', error); }); }; console.log('🛠️ Función debugRoutine() disponible globalmente'); console.log('💡 Uso: debugRoutine("ID_DE_RUTINA")'); })(); console.log('🔍 === VERIFICACIÓN DE IDs DE RUTINA COMPLETADA ==='); console.log('📋 Revisa los logs anteriores para identificar problemas con los IDs de rutina'); console.log('🛠️ Si encuentras IDs de rutina, puedes usar debugRoutine("ID") para verificar individualmente');
